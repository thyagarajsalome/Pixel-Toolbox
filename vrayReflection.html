V-Ray Reflection Blending Tool

Combine a V-Ray reflection pass with up to 500 main images. Choose a blending mode, adjust opacity, and download your final images in one click.

How to Use This Web App

1. Upload your V-Ray reflection image (JPG or PNG).
2. Upload one or more main images (up to 500).
3. Select a blending mode that suits reflection (e.g. Screen, Add).
4. Adjust the reflection opacity as desired and preview the first image result.
5. Click "Blend Images" to process all main images.
6. Download all results as a ZIP file containing JPG/PNG files with the same filenames as originally uploaded.
